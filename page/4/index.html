开始健身(2017.03)

今年的小目标是坚持健身一年。首先，我对于健身并没有那么痴迷，相反对于网上那些“不健身就怎么怎么样”，“健身多高大上”的帖子也挺反感的。我之所以健身，是想让自己的生活变得更加自律，顺便收获个好的身材也挺好的。其次，自从开始上班后感觉时间过得特别快，每天的生活都挺无聊，也想通过健身增加点生活的复杂度。人们总是高估自己短时间内能达到的目标，而低估长时间坚持而能做到的事情，我想让一年后的自己看起来比现在更好，所以坚持吧，少年！

DVR

2017-01-07

什么是DVR？

DVR，全称为Digital Video Recorder，即数字视频录像机。本文描述的DVR以车载DVR为主，如：行车记录仪，智能后视镜等。

DVR基本功能

DVR的功能很简单，就是录制视频并保存到硬盘中。根据不同的应用场景，会有一些定制的功能。在智能后视镜上的DVR，就会有前后录制，回放，拍照，加锁，紧急录制等功能。

Git

2016-11-14

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

生成秘钥

  • ssh-keygen -t rsa
    生成秘钥后，将公钥id_rsa.pub发给Git服务器，push到仓库时就可以免密码了。

分支命令

  • git branch -a 列出本地分支和远程分支
  • git branch -b 创建分支并切换
  • git branch -d 删除本地分支
  • git checkout 分支名 切换分支

二十一

2015-11-16

记得在《黄金时代》里，王小波这样说过，“那一天我二十一岁，在我一生的黄金时代。我有好多奢望。我想爱，想吃，还想在一瞬间变成天上半明半暗的云。后来我才知道，生活就是个缓慢受锤的过程，人一天天老下去，奢望也一天天消失，最后变得像挨了锤的牛一样。可是我过二十一岁生日时没有预见到这一点。我觉得自己会永远生猛下去，什么也锤不了我”。
