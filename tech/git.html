Git

2016-11-14

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

生成秘钥

  • ssh-keygen -t rsa
    生成秘钥后，将公钥id_rsa.pub发给Git服务器，push到仓库时就可以免密码了。

分支命令

  • git branch -a 列出本地分支和远程分支
  • git branch -b 创建分支并切换
  • git branch -d 删除本地分支
  • git checkout 分支名 切换分支

回退到某个版本

  • git reset –hard 版本号 回退到某个版本

比较两个版本的不同

  • git diff 版本号 版本号 –stat

与远程分支绑定

  • git branch –set-upstream-to main/master 与远程分支绑定
  • git branch –track [branchName] [remoteName] 创建分支并与远程分支绑定

保存工作现场

当需要pull代码时，此时的更改不想commit，可以将其保存到stash栈中，此时版本会回到最近一次提交后的样子。当需要恢复工作现场时，可以将保存的工作现场弹出栈，主要用到以下命令：

  • git stash save “保存的名称” 保存工作现场，压入stash栈中。
  • git stash list 查看stash栈信息
  • git stash pop ，git stash apply 弹出工作现场（区别：pop会将工作现场弹出栈后删除stash信息，apply则会保留）
  • git stash clear 将栈清空

临时调试

当在主分支添加修改功能，如果调试会将代码变动较大，修改麻烦，可以临时创建一个分支，并回退到适合调试的版本，完成功能，然后可以将修改的文件把主分支的该文件替换掉。

  • git checkout 临时分支 – 文件

让单个文件回退到指定版本

  • git log xxx.cpp
  • git reset 版本号 xxx.cpp
  • git commit -m’reset’
  • git checkout xxx.cpp

追踪某个文件的修改记录

  • git log -p fileName

远程仓库

  • git remote -v 列出远程仓库
  • git remote add repo XXX.git 添加远程仓库
  • git remote rename repo newrepo 重命名远程仓库
  • git remote rm XXX.git 删除远程仓库

忽略提交文件

当不想将某个文件或文件夹随项目一起提交时，可以在.gitignore中添加忽略文件
如果添加之前已经有提交，那么添加无效，因为改文件已经被跟踪。
解决方法：删除缓存 git rm -r –cached .
